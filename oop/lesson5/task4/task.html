Descriptor - a class that stores and controls the attributes of other classes. In general, any class that implements one of the special methods - __get__, __set__, __delete__, is a descriptor.


Example:
class ExternalStorage:
     __slots__ = ("attribute_name",)
     __storage = {}

     def __init __ (self, attribute_name):
         self.attribute_name = attribute_name

     def __set __ (self, instance, value):
         self .__ storage [id (instance), self.attribute_name] = value

     def __get __ (self, instance, owner = None):
         if instance is None:
             return self
         return self .__ storage [id (instance), self.attribute_name]

class Point:
     __slots__ = ()
     x = ExternalStorage ("x")
     y = ExternalStorage ("y")
     def __init __ (self, x = 0, y = 0):
         self.x = x
         self.y = y

P1 = Point (1,2)
P2 = Point (3,4)

In this case, the Point class does not have its own attributes x, y, though their causes, as if they are - in fact, they are stored in the descriptor ExternalStorage.
