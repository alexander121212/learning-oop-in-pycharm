Good news!
We can add attributes into object of the class or instance in our code not only in class!!!
For example:

class Student:
     """this class describes
     the average student"""

     debts_count = 10

     def pass_one_debt(self):
         self.debts_count -= 1

     def pass_all_debts(self):
         self.debts_count = 0
Student.faculty = "FIB"
now, object Student has attribute faculty and all instances were created after this will have this attribute
