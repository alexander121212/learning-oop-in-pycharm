Both are attributes of class. The real answer lay in understanding the distinction between class and instance attributes.

Python classes and instances of classes each have their own distinct namespaces represented by pre-defined attributes Student.__dict__ and instance_of_Student.__dict__, respectively.

When you try to access an attribute from an instance of a class, it first looks at its instance namespace. If it finds the attribute, it returns the associated value. If not, it then looks in the class namespace and returns the attribute (if itâ€™s present, throwing an error otherwise).
