Python provides 2 ways remove attributes of the class
First is operator del:
del ClassName.attribute
Second is function delattr(obj, 'attribute')
del is more explicit and efficient;
delattr allows dynamic attribute deleting.
Why?
del ClassName.attribute compiles to two bytecode instructions:
0 LOAD_FAST 0 (ClassName)
3 DELETE_ATTR 0 (attribute)
whereas delattr(ClassName, "attribute") takes five:
0 LOAD_GLOBAL 0 (delattr)
3 LOAD_FAST 0 (ClassName)
6 LOAD_CONST 1 ('attribute')
9 CALL_FUNCTION 2
12 POP_TOP
