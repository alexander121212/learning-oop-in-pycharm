What the difference between bound method and function?

class Student:
     """this class describes
     the average student"""
    
     debts_count = 10
    
     def __init__(self, desire_to_learn=True):
     self.desire_to_learn = desire_to_learn
    
     def pass_one_debt(self):
     self.debts_count -= 1

student = Student()
print(student.pass_one_debt)
print(Student.pass_one_debt)

this code print
<bound method Student.pass_one_debt of <__main__.Student object at 0x7f7679568358>>
<function Student.pass_one_debt at 0x7f76795637b8>

A function is created by the def statement, or by lambda. When a function appears within the body of a class statement (or is passed to a type class construction call), it is transformed into an unbound method. When a function is accessed on a class instance, it is transformed into a bound method, that automatically supplies the instance to the method as the first self parameter.

In python 3 unbound method is missing. And when a function appears within the body of a class statement, it is function type now
