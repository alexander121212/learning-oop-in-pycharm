Python classes may have attributes or not.
For definition attributes you need write variable in body of class.
For example:
class Student:
    """this class describes
     the average student"""

     subject_count = 10

if you want to know is attributes in class, you can use hasattr() function.
For example, hasattr(Student, "subjectCount") give True, because our class
contain attributes subjectCount.
