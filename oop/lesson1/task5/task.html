What does it mean class?

Сlass - is a template , a description not only of the structure of the data ( what it can or should be ) , but also behavior , functionality of the objects of this class. To understand what the object of the class must be imagined abstracting and understand what the object is. Philosophy says that the object - a philosophical category expressing something that exists in reality . Programming says that the object - it is an entity in the virtual space , has a certain status and behavior , has given values ​​of properties ( attributes) and operations on them ( methods). Now we learn programming, so let's build on the second definition. So, we have object which has status and behavior and we have class that can describe status and behavior in terms of attributes and methods. Therefore , the class object - an object state and behavior which describes the class Instance is specific object of this class.
